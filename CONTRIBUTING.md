# Contributing

Contributions are welcome.

Good contributions include:

- Better audit rubric wording
- More examples
- More prompts for specific industries
- Claude Code skill improvements
- Codex `AGENTS.md` improvements
- Lightweight scripts for URL/page extraction
- Evals that catch weak audit outputs

## Rules

1. Do not add fake testimonials, fake metrics, or fake claims.
2. Do not make the project appear official or endorsed by Nik Sharma or Sharma Brands.
3. Keep instructions practical for coding agents.
4. Prefer clear examples over vague advice.
5. Keep skill descriptions concise enough for agent systems that limit metadata length.

## Development

Run the URL scanner:

```bash
npm run audit -- https://example.com
```
