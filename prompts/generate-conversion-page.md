# Prompt: Generate a New Conversion Page

Build a conversion-focused website or landing page using the Sharma Brands-inspired framework.

Business: [describe business]
Audience: [describe buyer]
Primary conversion goal: [demo booking / WhatsApp inquiry / checkout / lead form / signup]
Offer: [describe offer]
Proof assets available: [logos/testimonials/screenshots/case studies]
Pricing: [known pricing or say unknown]
Brand style: [premium/minimal/playful/clinical/etc.]
Tech stack: [Next.js/React/Vite/etc.]

Required sections:

1. Hero
2. Trust bar
3. Problem
4. Solution workflow
5. Benefits
6. Product/service demo
7. Use cases
8. Proof
9. Pricing/offer
10. FAQs
11. Final CTA
12. Footer

Rules:

- Make the offer clear within 3 seconds.
- Build mobile-first.
- Use specific CTA copy.
- Include proof placeholders if real proof is unavailable.
- Do not invent testimonials, logos, numbers, guarantees, pricing, or case studies.
- Add SEO metadata and Open Graph tags.
- Add analytics placeholders.
- Keep content easy to edit.
