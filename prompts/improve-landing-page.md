# Prompt: Improve a Landing Page

Improve this landing page using the Sharma Brands-inspired website auditing skill.

Before editing:

1. Run a short audit.
2. Identify the biggest conversion blockers.
3. State the proposed changes.

Then update the page to improve:

- 3-second clarity
- Hero headline/subheadline
- CTA specificity
- Trust/proof placement
- Buyer education
- Mobile readability
- FAQ quality
- Offer/pricing clarity

Do not invent testimonials, logos, metrics, pricing, guarantees, or case studies.
Use placeholders where proof is missing.
