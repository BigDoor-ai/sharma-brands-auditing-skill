# Prompt: Audit a Website Codebase

Audit this website codebase using the Sharma Brands-inspired conversion framework.

Review relevant files/components, especially:

- Hero
- CTA components
- Trust/proof sections
- Pricing/offer sections
- FAQs
- Forms
- SEO metadata
- Mobile layout

Return:

1. Overall score
2. Main conversion blockers
3. Files/components to change
4. Suggested copy improvements
5. Priority fix list

Do not invent testimonials, logos, metrics, pricing, guarantees, or case studies.
Use placeholders where proof is missing.
